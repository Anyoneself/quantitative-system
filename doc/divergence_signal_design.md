# 上涨背驰与下跌背驰信号设计

## 1. 定位

背驰用于判断：

```text
价格还在创新高或创新低，但推动价格的力量是否已经变弱。
```

本系统中，背驰信号应成为股票买入和卖出判断中的核心结构依据之一。

它不直接承诺反转，但可以帮助回答两个关键问题：

```text
下跌趋势是否已经接近衰竭，是否值得进入买入观察？
上涨趋势是否已经动能衰减，是否需要止盈或减仓？
```

## 2. 名词约定

为了避免混淆，本文使用以下定义：

```text
下跌背驰：价格创新低，但动能不再同步创新低。常用于寻找底部反弹或一买/二买候选。
上涨背驰：价格创新高，但动能不再同步创新高。常用于识别顶部风险或止盈候选。
```

有些资料会把“上涨背驰”叫“顶背驰”，把“下跌背驰”叫“底背驰”。系统中建议统一输出：

```text
上涨背驰候选 / 下跌背驰候选
```

同时兼容显示：

```text
顶背驰候选 = 上涨背驰候选
底背驰候选 = 下跌背驰候选
```

## 3. 为什么背驰重要

### 3.1 下跌背驰对买入的意义

下跌背驰说明：

```text
价格还在跌，但下跌动能已经变弱。
```

这通常意味着：

- 空头力量正在衰减。
- 抛压可能接近释放尾声。
- 价格可能进入止跌、横盘、反弹或趋势修复阶段。

因此，下跌背驰应作为买入观察的重要依据。

但它不能单独触发买入，必须配合：

- 低位十字星或长下影线。
- 成交量不再放大下跌。
- MACD 柱体收敛或金叉。
- 主力资金不再持续流出，最好开始净流入。
- 缠论底分型、二买候选或中枢下沿不再跌破。

### 3.2 上涨背驰对卖出的意义

上涨背驰说明：

```text
价格还在涨，但上涨动能已经变弱。
```

这通常意味着：

- 追高风险上升。
- 上涨可能进入末端。
- 已持仓时应考虑止盈或减仓。

因此，上涨背驰应作为卖出风险的重要依据。

## 4. 可用于背驰判断的现有数据

系统当前已经具备这些字段：

```text
open / high / low / close
volume / amount
MA5 / MA10 / MA20 / MA250
MACD DIF / DEA / MACD 柱
缠论分型 / 笔 / 中枢
资金流向
```

第一版背驰判断应优先基于：

- 价格新高 / 新低。
- MACD 柱体强弱。
- MACD DIF 强弱。
- 成交量变化。
- 缠论笔力度。
- 资金流方向。

## 5. 下跌背驰候选识别

### 5.1 基础条件

下跌背驰候选需要满足：

```text
当前价格创近 N 日新低
但 MACD 柱体没有同步创近 N 日新低
或 DIF 没有同步创近 N 日新低
```

建议第一版使用：

```text
N = 20 或 30
```

### 5.2 MACD 下跌背驰

规则：

```text
当前 close <= 近 20 日最低 close
且 当前 MACD 柱 > 前一轮低点附近的 MACD 柱
```

含义：

```text
价格创新低，但下跌动能不再创新低。
```

### 5.3 DIF 下跌背驰

规则：

```text
当前 close <= 近 20 日最低 close
且 当前 DIF > 前一轮低点附近的 DIF
```

含义：

```text
价格更低，但 MACD 快线没有更弱。
```

### 5.4 量能下跌背驰

规则：

```text
当前 close 创新低
但 下跌日成交量没有继续放大
或 成交量 / 5 日均量低于上一轮下跌
```

含义：

```text
继续下跌时抛压没有同步增强。
```

### 5.5 资金流下跌背驰

规则：

```text
价格创新低
但 主力净流出金额减少
或 5 日主力净流出转为收敛
或 当日主力开始净流入
```

含义：

```text
价格继续跌，但资金流出压力正在减弱。
```

## 6. 上涨背驰候选识别

### 6.1 基础条件

上涨背驰候选需要满足：

```text
当前价格创近 N 日新高
但 MACD 柱体没有同步创近 N 日新高
或 DIF 没有同步创近 N 日新高
```

建议第一版使用：

```text
N = 20 或 30
```

### 6.2 MACD 上涨背驰

规则：

```text
当前 close >= 近 20 日最高 close
且 当前 MACD 柱 < 前一轮高点附近的 MACD 柱
```

含义：

```text
价格创新高，但上涨动能没有同步创新高。
```

### 6.3 DIF 上涨背驰

规则：

```text
当前 close >= 近 20 日最高 close
且 当前 DIF < 前一轮高点附近的 DIF
```

含义：

```text
价格更高，但 MACD 快线没有更强。
```

### 6.4 量能上涨背驰

规则：

```text
价格创新高
但 成交量没有同步放大
或 成交量 / 5 日均量低于上一轮上涨
```

含义：

```text
继续上涨缺乏新的资金承接。
```

### 6.5 资金流上涨背驰

规则：

```text
价格创新高
但 主力净流入减少
或 主力资金转为净流出
```

含义：

```text
价格还在涨，但主力资金已经不再积极推动。
```

## 7. 建议输出结构

建议新增模块：

```text
src/analysis/divergence.py
```

建议输出：

```python
@dataclass(frozen=True)
class DivergenceResult:
    bullish_divergence: bool
    bearish_divergence: bool
    bullish_score_adjustment: int
    bearish_risk_adjustment: int
    signal: str
    confidence: int
    reasons: list[str]
```

字段说明：

- `bullish_divergence`：下跌背驰候选，对买入有支持。
- `bearish_divergence`：上涨背驰候选，对卖出有支持。
- `bullish_score_adjustment`：买入评分调整。
- `bearish_risk_adjustment`：卖出风险调整。
- `signal`：中文信号名。
- `confidence`：置信度，0-100。
- `reasons`：触发原因。

## 8. 买入评分接入

### 8.0 5/10 日均线缠绕系统

原始表述中关于“5 日均线和 10 日均线位置关系”的内容，可以抽象成一个短周期均线缠绕系统。

系统定义：

```text
多头排列：MA5 > MA10
空头排列：MA5 < MA10
均线缠绕：MA5 与 MA10 接近并反复交叉或贴合
```

均线缠绕只有两种演化结果：

```text
中继：缠绕结束后延续原来的均线排列方向
转折：缠绕结束后改变原来的均线排列方向
```

因此，操作上只关注两类买点：

```text
空头排列后的最后一次缠绕 + 下跌背驰 -> 底部买点
多头排列后的第一次缠绕 + 中继确认 -> 加仓或二次买点
```

### 8.0.1 均线体位判断

第一步先判断 MA5 和 MA10 的位置：

```text
MA5 > MA10：短期多头排列
MA5 < MA10：短期空头排列
MA5 与 MA10 反复贴合或交叉：均线缠绕
```

均线缠绕本身不是买点，它只是提示：

```text
短期趋势即将选择方向。
```

### 8.0.2 多头排列后的第一次缠绕

如果股票已经处在多头排列中：

```text
MA5 > MA10
```

之后第一次出现 MA5/MA10 缠绕，通常更可能是上涨中继，而不是趋势结束。

更可信的中继条件：

```text
缠绕前 MA5 上行有力
缠绕前价格涨幅不是过度透支
缠绕前成交量没有异常放大
缠绕时成交量温和收缩
缠绕后价格重新站上 MA5 / MA10
资金流没有持续净流出
MACD 没有死叉或柱体没有明显走弱
```

对应系统信号：

```text
多头排列 + 第一次均线缠绕 + 缩量回调 + MACD 未明显走弱 + 资金流未恶化
```

建议评分：

```text
买入评分 +6 到 +10
```

这类信号适合用于趋势股回调加仓，而不是用于抄底。

### 8.0.3 多头排列后的多次缠绕

如果多头排列后已经出现第三次或第四次均线缠绕，风险应提高。

原因是：

```text
趋势经过多次缠绕后，上涨中继概率下降，转折概率上升。
```

风险特征：

```text
MA5 上攻斜率变弱
价格创新高但 MACD 柱不创新高
资金流入减弱或主力净流出
高位放量滞涨
缠论出现上涨背驰候选
```

建议处理：

```text
买入评分不加分
卖出风险加分
如果同时出现上涨背驰，进入止盈或减仓观察
```

### 8.0.4 空头排列后的缠绕

如果股票处在空头排列中：

```text
MA5 < MA10
```

第一次均线缠绕通常不应直接视为底部转折。它更可能只是下跌中继或弱反弹。

更值得关注的是：

```text
空头排列经过多次缠绕后，出现下跌背驰和空头陷阱。
```

底部买点需要同时满足：

```text
空头排列末端
MA5 / MA10 多次缠绕后不再继续有效下跌
价格短暂创新低
MACD 柱或 DIF 不再创新低
成交量没有继续放大下跌
主力资金流出收敛或转流入
低位出现十字星、长下影或底分型
```

对应系统信号：

```text
空头排列末端 + 均线缠绕 + 下跌背驰 + 低位反弹 K 线 + 资金流改善
```

建议评分：

```text
买入评分 +15 到 +20
```

这是文中强调的第一类高性价比买点。

### 8.0.5 两个最值得关注的买点

根据该均线缠绕系统，真正值得重点关注的买点只有两个：

```text
买点一：空头排列末端，最后一次缠绕后出现下跌背驰。
买点二：多头排列确立后，第一次缠绕形成上涨中继。
```

系统中可以转化为：

```text
买点一 = MA5 < MA10 后期 + 多次缠绕 + 下跌背驰 + 资金流改善
买点二 = MA5 > MA10 初期 + 第一次缠绕 + 缩量回调 + 趋势未破坏
```

### 8.0.6 风险来源

这套规则的主要风险是误判：

```text
把下跌中继误判为底部转折
把上涨转折误判为上涨中继
把假背驰误判为有效背驰
```

因此系统不应只看 MA5/MA10 缠绕，而应要求至少和以下信号共振：

- MACD 背驰。
- 成交量衰竭或缩量回调。
- 资金流改善。
- 缠论底分型、二买或三买候选。
- 年线或 20 日均线位置修复。

第一版实现建议新增字段：

```text
ma5_above_ma10
ma5_ma10_weaving
ma5_ma10_weaving_count
ma5_ma10_first_bullish_weaving
ma5_ma10_late_bearish_weaving
```

这些字段可以作为背驰模块的前置结构条件，而不是单独作为买卖依据。

下跌背驰应作为买入评分的重要依据，但需要分层。

### 8.1 强下跌背驰

条件：

```text
价格创新低
MACD 柱底背驰
DIF 底背驰
资金流出收敛或转流入
低位 K 线出现十字星/长下影
```

建议加分：

```text
+15 到 +20
```

### 8.2 普通下跌背驰

条件：

```text
价格创新低
MACD 柱或 DIF 至少一个出现底背驰
```

建议加分：

```text
+8 到 +12
```

### 8.3 弱下跌背驰

条件：

```text
只有量能或资金流出现衰竭迹象
但 MACD 尚未确认
```

建议加分：

```text
+3 到 +6
```

## 9. 卖出评分接入

上涨背驰应作为卖出风险的重要依据。

### 9.1 强上涨背驰

条件：

```text
价格创新高
MACD 柱顶背驰
DIF 顶背驰
资金流入减弱或转流出
高位放量滞涨或长上影
```

卖出风险加分：

```text
+18 到 +25
```

### 9.2 普通上涨背驰

条件：

```text
价格创新高
MACD 柱或 DIF 至少一个出现顶背驰
```

卖出风险加分：

```text
+10 到 +16
```

### 9.3 弱上涨背驰

条件：

```text
只有量能或资金流出现衰竭迹象
但 MACD 尚未确认
```

卖出风险加分：

```text
+4 到 +8
```

## 10. 与缠论的关系

背驰和缠论结构应组合使用。

买入侧：

```text
下跌背驰 + 底分型 + 二买候选 -> 买入评分显著提高
下跌背驰 + 中枢下沿不破 -> 买入观察提高
下跌背驰 + 资金流转正 -> 买入信号更可信
```

卖出侧：

```text
上涨背驰 + 顶分型 -> 卖出风险提高
上涨背驰 + 顶背驰候选 -> 止盈风险提高
上涨背驰 + 跌回中枢 -> 趋势破坏风险提高
```

## 11. 与资金流的关系

资金流用于确认背驰是否更可信。

下跌背驰更可信的资金流状态：

```text
主力净流出减少
5 日主力净流出收敛
当日主力净流入
超大单开始净流入
```

上涨背驰更危险的资金流状态：

```text
主力净流入减少
主力连续净流出
价格创新高但超大单不再流入
小单追涨明显而主力流出
```

建议新增“背驰信号”板块。

展示字段：

```text

置信度：0-100
买入评分调整

触发依据
```

示例：

```text
背驰信号：下跌背驰候选
置信度：72
买入调整：+12
依据：价格创 20 日新低，但 MACD 柱和 DIF 未同步创新低，且主力流出收敛。
```

## 14. 最小可验收标准

第一版完成时，至少应满足：

- 可以识别价格创新低但 MACD 动能不创新低的下跌背驰候选。
- 可以识别价格创新高但 MACD 动能不创新高的上涨背驰候选。
- 下跌背驰可以提高买入评分。
- 上涨背驰可以提高卖出风险评分。
- 前端能展示背驰类型、置信度和依据。
- CLI 能输出背驰说明。
- 单元测试覆盖下跌背驰、上涨背驰、无背驰三类情况。

## 15. 风险说明

背驰不是反转保证。

使用时必须注意：

- 下跌背驰后仍可能继续下跌。
- 上涨背驰后仍可能继续上涨。
- 背驰需要等待价格确认，不能只看指标。
- 背驰最好和资金流、成交量、缠论结构共振后再提高权重。
- 所有背驰加减分都必须经过回测验证。
