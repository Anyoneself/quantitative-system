# 股票指标总览

本文档列举系统当前已经使用的股票指标、结构信号和机器学习特征，并说明 Bollinger Bands 等常用指标当前是否已经接入。

## 1. 指标使用范围

系统当前只使用公开日线行情数据，基础字段来自 `PriceVolumeBar`：

```text
trade_date
open
high
low
close
volume
amount
```

系统不会读取基本面、新闻、研报、资金流、盘口明细或分钟级数据。

当前分析分为四层：

```text
第一层：基础量价指标
第二层：量价形态识别
第三层：机器学习历史相似样本
第四层：缠论结构辅助
```

最终输出的是综合评分：

```text
综合评分 = 机器学习概率分 + 缠论结构调整分
```

综合评分不是收益承诺，只是辅助筛选和观察。

## 2. 当前已经使用的基础指标

### 2.1 收盘价

字段：`close`

用途：

- 作为当前价格基准。
- 用于计算涨跌幅、均线、突破位置和机器学习特征。
- 前端图表绘制价格曲线。

### 2.2 1 日涨跌幅

字段：`return_1d`

公式：

```text
1 日涨跌幅 = (当前收盘价 - 前一交易日收盘价) / 前一交易日收盘价
```

用途：

- 判断当日上涨、下跌、涨停、跌停。
- 判断放量下跌、高位滞涨、缩量回调等形态。
- 作为机器学习特征之一。

### 2.3 5 日涨跌幅

字段：`return_5d`

公式：

```text
5 日涨跌幅 = (当前收盘价 - 5 日前收盘价) / 5 日前收盘价
```

用途：

- 衡量短期价格强弱。
- 作为机器学习特征之一。

### 2.4 20 日涨跌幅

字段：`return_20d`

公式：

```text
20 日涨跌幅 = (当前收盘价 - 20 日前收盘价) / 20 日前收盘价
```

用途：

- 判断近 20 日是否涨幅过高。
- 识别追高风险。
- 作为机器学习特征之一。

### 2.5 5 日均线

字段：`ma5`

公式：

```text
MA5 = 最近 5 个交易日收盘价平均值
```

用途：

- 衡量短期趋势。
- 机器学习中使用“收盘价相对 5 日均线位置”。

### 2.6 10 日均线

字段：`ma10`

公式：

```text
MA10 = 最近 10 个交易日收盘价平均值
```

用途：

- 当前作为基础指标输出字段保留。
- 暂未直接参与评分规则。

### 2.7 20 日均线

字段：`ma20`

公式：

```text
MA20 = 最近 20 个交易日收盘价平均值
```

用途：

- 判断价格是否站上 20 日均线。
- 判断短中期趋势是否偏强。
- 作为机器学习特征之一。
- 用于生成风险提示和后续观察点。

### 2.8 前一日 20 日均线

字段：`ma20_prev`

公式：

```text
ma20_prev = 前一交易日视角下的 20 日均线
```

用途：

- 判断 20 日均线是否向上。
- 作为机器学习特征中的“20 日均线斜率”。

### 2.9 20 日前高

字段：`high_20d_prev`

公式：

```text
high_20d_prev = 当前交易日前 20 个交易日的最高价
```

用途：

- 判断是否放量突破前 20 日高点。
- 计算当前收盘价距离 20 日高点的位置。
- 作为机器学习特征之一。

### 2.10 20 日前低

字段：`low_20d_prev`

公式：

```text
low_20d_prev = 当前交易日前 20 个交易日的最低价
```

用途：

- 判断当前价格距离短期低位的位置。
- 用于数据证据展示。

### 2.11 当前成交量

字段：`volume`

用途：

- 判断当日量能强弱。
- 和均量比较形成量比。
- 用于识别放量突破、缩量回调、放量下跌等形态。

### 2.12 5 日均量

字段：`volume_ma5`

公式：

```text
volume_ma5 = 当前交易日前 5 个交易日成交量平均值
```

用途：

- 作为短期量能基准。
- 用于计算成交量 / 5 日均量。

### 2.13 20 日均量

字段：`volume_ma20`

公式：

```text
volume_ma20 = 当前交易日前 20 个交易日成交量平均值
```

用途：

- 作为中期量能基准。
- 用于计算成交量 / 20 日均量。

### 2.14 成交量 / 5 日均量

字段：`volume_ratio_5d`

公式：

```text
volume_ratio_5d = 当前成交量 / 5 日均量
```

用途：

- 小于 `0.8` 时视为缩量。
- 大于 `1.5` 时视为放量。
- 参与涨停缩量、放量突破、缩量回调、放量下跌等形态识别。
- 作为机器学习特征之一。

### 2.15 成交量 / 20 日均量

字段：`volume_ratio_20d`

公式：

```text
volume_ratio_20d = 当前成交量 / 20 日均量
```

用途：

- 衡量当前量能相对中期水平是否异常。
- 作为机器学习特征之一。
- Web 前端摘要展示。

### 2.16 成交额

字段：`amount`

用途：

- 当前作为基础行情字段保留。
- 更常用的是 20 日平均成交额。

### 2.17 20 日平均成交额

字段：`amount_ma20`

公式：

```text
amount_ma20 = 当前交易日前 20 个交易日成交额平均值
```

用途：

- 判断基础流动性。
- 小于 `5000 万元` 时标记为流动性偏低。

### 2.18 涨停判断

字段：`is_limit_up`

规则：

```text
1 日涨跌幅 >= 9.8%
```

用途：

- 配合成交量判断“涨停缩量”。

### 2.19 跌停判断

字段：`is_limit_down`

规则：

```text
1 日涨跌幅 <= -9.8%
```

用途：

- 当前作为基础风险字段保留。

### 2.20 年线 MA250

字段：`ma250`

公式：

```text
MA250 = 最近 250 个交易日收盘价平均值
```

用途：

- 判断长期趋势是否修复。
- 判断股票是否已经站上年线。
- 参与“强势股回档”策略评分。

### 2.21 前一日年线 MA250

字段：`ma250_prev`

公式：

```text
ma250_prev = 前一交易日视角下的 250 日均线
```

用途：

- 判断年线是否走平后向上拐头。
- 年线向上时作为长期趋势修复加分项。

### 2.22 新股可用首日高点

字段：`first_day_high`

规则：

```text
如果可用日线不足 250 个交易日，则取当前可用数据第一天的最高价
如果可用日线达到 250 个交易日，则该字段为 0，表示不按新股首日高点处理
```

用途：

- 用于识别“放量突破上市初期首日高点”的新股强势信号。
- 注意：公开接口只返回可用窗口内的数据，无法保证一定是真正上市首日。

### 2.23 当日 K 线结构

字段：

```text
open
high
low
close
```

用途：

- 判断十字星。
- 判断长下影线。
- 判断低位止跌反弹信号。
- 辅助买入评分和卖出风险评分。

## 3. 当前已经使用的量价形态

### 3.1 涨停缩量

字段：`limit_up_low_volume`

规则：

```text
涨停
且 成交量 / 5 日均量 < 0.8
```

含义：

```text
价格强势上涨，但成交量没有明显放大，说明卖压可能暂时较小。
```

### 3.2 放量突破

字段：`breakout_with_volume`

规则：

```text
收盘价 > 前 20 日高点
且 成交量 / 5 日均量 > 1.5
```

含义：

```text
价格突破短期高点，同时有成交量配合，突破有效性相对更强。
```

### 3.3 缩量回调

字段：`pullback_low_volume`

规则：

```text
-4% <= 1 日涨跌幅 < 0
且 成交量 / 5 日均量 < 0.8
且 收盘价 > 20 日均线
```

含义：

```text
价格小幅回调但量能收缩，且仍在 20 日均线上方，抛压相对可控。
```

### 3.4 放量大跌

字段：`high_volume_drop`

规则：

```text
1 日涨跌幅 <= -5%
且 成交量 / 5 日均量 > 1.5
```

含义：

```text
价格明显下跌且成交量放大，短线卖压较强。
```

### 3.5 高位放量滞涨

字段：`high_volume_stagnation`

规则：

```text
abs(1 日涨跌幅) <= 2%
且 成交量 / 5 日均量 > 1.5
且 20 日涨跌幅 > 30%
```

含义：

```text
前期已经大幅上涨，当前放量但价格涨不动，可能存在筹码松动。
```

### 3.6 价涨量缩

字段：`price_up_volume_down`

规则：

```text
1 日涨跌幅 > 0
且 成交量 / 5 日均量 < 0.8
```

含义：

```text
价格上涨但量能不足，后续需要确认资金是否继续跟进。
```

### 3.7 价跌量增

字段：`price_down_volume_up`

规则：

```text
1 日涨跌幅 < 0
且 成交量 / 5 日均量 > 1.5
```

含义：

```text
价格下跌且量能放大，卖压上升。
```

### 3.8 站上 20 日均线

字段：`close_above_ma20`

规则：

```text
收盘价 > 20 日均线
```

含义：

```text
短中期趋势相对偏强。
```

### 3.9 20 日均线向上

字段：`ma20_up`

规则：

```text
当前 20 日均线 > 前一日 20 日均线
```

含义：

```text
中期趋势有支撑。
```

### 3.10 流动性偏低

字段：`low_liquidity`

规则：

```text
20 日平均成交额 > 0
且 20 日平均成交额 < 5000 万元
```

含义：

```text
成交额偏低，可能出现买卖滑点、成交困难或价格跳动较大。
```

### 3.11 20 日涨幅过高

字段：`high_20d_return`

规则：

```text
20 日涨跌幅 > 30%
```

含义：

```text
短期累计涨幅较高，继续追高的回撤风险上升。
```

### 3.12 放量跌破 20 日均线

字段：`volume_drop_below_ma20`

规则：

```text
1 日涨跌幅 < -2%
且 收盘价 < 20 日均线
且 成交量 / 5 日均量 > 1.5
```

含义：

```text
价格跌破中期趋势线且放量，趋势转弱风险较高。
```

### 3.13 放量突破年线

字段：`yearline_breakout_with_volume`

规则：

```text
收盘价 > 年线 MA250
且 年线 MA250 > 前一日年线 MA250
且 1 日涨跌幅 > 0
且 成交量 / 5 日均量 > 1.5
```

含义：

```text
价格放量站上长期趋势线，且年线开始向上，长期趋势可能进入修复阶段。
```

评分影响：加分。

### 3.14 缩量回踩年线

字段：`yearline_pullback_low_volume`

规则：

```text
收盘价 > 年线 MA250
且 收盘价 <= 年线 MA250 * 1.04
且 -5% <= 1 日涨跌幅 < 0
且 成交量 / 5 日均量 < 0.8
```

含义：

```text
股票已经站上年线，回调时缩量且没有明显跌破年线，符合“强势股回档观察”的思路。
```

评分影响：加分。

### 3.15 年线向上拐头

字段：`ma250_turning_up`

规则：

```text
年线 MA250 > 前一日年线 MA250
```

含义：

```text
长期趋势开始向上修复，比单纯站上年线更有趋势意义。
```

评分影响：加分。

### 3.16 仍在年线下方

字段：`close_above_ma250`

规则：

```text
收盘价 > 年线 MA250
```

如果该条件不成立，说明股票仍在年线下方。

含义：

```text
长期趋势尚未修复，按强势股回档策略应先等待重新站上年线。
```

评分影响：扣分。

### 3.17 新股放量突破首日高点

字段：`first_day_high_breakout_with_volume`

规则：

```text
可用首日高点 > 0
且 收盘价 > 可用首日高点
且 成交量 / 5 日均量 > 1.5
```

含义：

```text
新股或次新股突破上市初期压力位，且有成交量配合，说明市场可能重新定价。
```

评分影响：加分。

### 3.18 低位十字星

字段：`doji_reversal`

规则：

```text
实体 / 全日振幅 <= 18%
且 收盘价接近 20 日低点
且 最近 5 日或 20 日处于回调状态
```

含义：

```text
低位附近出现多空分歧，可能进入止跌观察区。
```

评分影响：小幅加分。

### 3.19 长下影线

字段：`long_lower_shadow`

规则：

```text
下影线 / 全日振幅 >= 45%
且 下影线明显长于上影线
```

含义：

```text
盘中下探后出现承接，短线抛压可能缓和。
```

评分影响：小幅加分；在卖出判断中降低一部分卖出风险。

### 3.20 低位止跌反弹

字段：`low_position_rebound`

规则：

```text
收盘价接近 20 日低点
且 出现低位十字星或长下影线
且 当日跌幅不超过 3%
```

含义：

```text
股价在短期低位附近出现止跌迹象，可以进入反弹观察，但仍需要后续放量站回均线确认。
```

评分影响：买入评分小幅加分；卖出风险小幅降低。

## 4. 当前已经使用的机器学习特征

机器学习模型使用最近约一年日线数据构造历史样本。每个样本使用 20 日窗口计算 9 个特征，并用“下一交易日是否上涨”作为标签。

当前特征包括：

| 序号 | 特征 | 含义 |
| --- | --- | --- |
| 1 | 1 日涨跌幅 | 当前短线涨跌状态 |
| 2 | 5 日涨跌幅 | 短期强弱 |
| 3 | 20 日涨跌幅 | 中期强弱 |
| 4 | 收盘价相对 5 日均线位置 | 是否偏离短期均线 |
| 5 | 收盘价相对 20 日均线位置 | 是否站上中期均线 |
| 6 | 20 日均线斜率 | 中期趋势方向 |
| 7 | 成交量 / 5 日均量 | 短期量能变化 |
| 8 | 成交量 / 20 日均量 | 中期量能变化 |
| 9 | 收盘价相对过去 20 日高点位置 | 是否接近或突破短期高点 |

年线 MA250、首日高点和强势回档形态当前作为规则评分使用，暂未进入机器学习特征向量。

当前支持五种模型：

- 稳健集成模型（5日）。
- 时间衰减 KNN（5日）。
- KNN 相似样本。
- 加权 KNN 相似样本。
- 逻辑回归。

## 5. 当前已经使用的 MACD 指标

MACD 使用收盘价序列计算，用于判断趋势动能变化，并参与买入评分和卖出风险评分。

当前字段：

```text
macd_dif
macd_dea
macd_histogram
macd_dif_prev
macd_dea_prev
macd_histogram_prev
```

公式：

```text
DIF = EMA12 - EMA26
DEA = DIF 的 9 日 EMA
MACD 柱 = 2 * (DIF - DEA)
```

当前识别：

- MACD 金叉。
- 零轴上方 MACD 金叉。
- MACD 死叉。
- MACD 柱体扩大。
- MACD 柱体走弱。

评分影响：

- 零轴上方金叉：买入评分加分。
- 普通金叉：买入评分小幅加分。
- MACD 柱体扩大：买入评分小幅加分。
- MACD 死叉：买入评分扣分，卖出风险加分。
- MACD 柱体走弱：买入评分扣分，卖出风险加分。

注意：MACD 不单独决定买卖，只作为趋势动能因子，需要结合价格位置、资金流、成交量和缠论结构。

## 6. 当前已经使用的缠论结构指标

缠论结构使用最近 90 个交易日，输出结构调整分并参与综合评分。

当前使用：

- 顶分型。
- 底分型。
- 上涨笔。
- 下跌笔。
- 最近中枢。
- 结构状态：上升结构、下降结构、盘整结构或不确定。
- 当前位置：中枢内、中枢上沿附近、中枢下沿附近、离开中枢上方、跌破中枢下方等。
- 买点候选：二买候选、三买候选或无。
- 风险结构：顶背驰候选、跌破中枢候选、下降结构延续、高位缩量候选等。

缠论结构调整分规则见 [缠论在系统中的使用设计](chan_theory_usage.md)。

## 7. 尚未接入但常见的技术指标

以下指标是股票分析中常见的技术指标，但当前代码尚未接入。本文档先列出它们，便于后续扩展时统一口径。

### 7.1 Bollinger Bands / BOLL / 布林线

当前状态：未接入。

常见组成：

```text
中轨 = N 日移动平均线，常用 20 日均线
上轨 = 中轨 + K * N 日标准差，常用 K=2
下轨 = 中轨 - K * N 日标准差，常用 K=2
```

常见用途：

- 判断价格是否接近波动区间上沿或下沿。
- 判断波动率扩张或收缩。
- 配合趋势判断突破是否有效。

未来接入建议：

- 使用 `boll_mid`、`boll_upper`、`boll_lower`、`boll_width`、`boll_percent_b`。
- 将 `boll_width` 作为波动率特征。
- 将收盘价相对上下轨位置作为追高或低位修复参考。

### 7.2 KDJ

当前状态：未接入。

常见用途：

- 衡量短期超买超卖。
- 辅助判断短线转折。

未来接入建议：

- 只作为短线参考，不直接作为买入核心信号。
- 避免在趋势行情里单独使用 KDJ 超买/超卖做反向判断。

### 7.3 RSI

当前状态：未接入。

常见用途：

- 衡量上涨和下跌力度的相对强弱。
- 辅助判断超买超卖和动能衰减。

未来接入建议：

- 可使用 `rsi6`、`rsi14`。
- 与 BOLL、MACD 一起作为风险辅助，而不是单独给买卖结论。

### 7.4 ATR

当前状态：未接入。

常见用途：

- 衡量真实波动范围。
- 用于动态止损、仓位控制和风险估算。

未来接入建议：

- 使用 `atr14` 估算短期波动。
- 用 ATR 辅助生成止损距离和仓位建议。

### 7.5 OBV

当前状态：未接入。

常见用途：

- 观察成交量是否随价格方向累积。
- 辅助判断资金是否持续流入或流出。

未来接入建议：

- 用 OBV 斜率作为量能趋势特征。
- 和价格趋势背离时作为风险提示。

## 7. 指标和决策的关系

系统不会因为单个指标触发就直接给出买入建议。

当前决策逻辑更接近：

```text
基础量价指标 -> 量价形态 -> 机器学习概率 -> 缠论结构调整 -> 综合评分和解释
```

使用时应优先看：

- 综合评分是否足够高。
- 历史相似上涨占比是否有优势。
- 缠论结构是否处在二买、三买或风险位置。
- 风险提示是否出现放量下跌、跌破均线、跌破中枢、顶背驰等信号。
- 成交额是否足够，避免流动性太差。

如果后续接入 BOLL、KDJ、RSI 等指标，也应先通过回测验证它们是否能提升胜率、收益或降低回撤，再进入正式评分。
